Does the score work? (forward validation)

Historical option data isn't available on these feeds, so this measures the score's edge going forward: it snapshots top-scoring contracts with their entry price, then re-quotes them over the following days and records the realized option return. If the score has edge, higher-score buckets should show better average returns. This needs days to weeks of data to mean anything - one snapshot proves nothing. It runs automatically once per market day while the server is up (snapshot + outcome update); the buttons below are for on-demand runs.
